# Emotion Detector Application

This is a Python-based web application developed using the Watson NLP library and Flask framework. The application analyzes user-provided text and detects various emotions such as anger, disgust, fear, joy, and sadness.

## Features
- AI-based emotion detection using Watson NLP.
- Formatted JSON output.
- Web interface built with Flask.
- Robust error handling for blank inputs.
- High-quality code compliant with static code analysis (Pylint).
